# 🧩 Crossword Solver using Minimax Algorithm

## 📌 Overview
This project implements a Crossword Puzzle Solver using the Minimax algorithm.  
The goal is to intelligently fill a crossword grid by selecting optimal word placements based on game constraints.

---

## 🧠 Algorithm Used
- **Minimax Algorithm**
  - Used for decision making
  - Simulates possible moves and chooses the optimal one
  - Assumes worst-case opponent scenario

---

## 🚀 Features
- Solves crossword puzzles programmatically  
- Uses AI-based decision making  
- Handles constraints like:
  - Word placement
  - Grid matching
  - Valid word selection

## 📊 How It Works
- The crossword grid is treated as one state
- Possible word placements will be generated
- Minimax evaluates each possibility
- The best move is selected based on an optimal outcome after preprocessings
